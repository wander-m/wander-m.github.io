JAVA注释、输入输出、项目迁移、调试程序

注释

  • 单行注释
    • 以“//”开始，没有结束符号，用于说明文字较少时
  • 多行注释
    • 以“/*”开始，以“*/”结束，用于说明文字较多时
  • 文档注释
    • 以“/**”开始,以“*/”结束，用于说明文字较多时
    • 该类注释允许在程序中嵌入关于程序的信息。
    • 可以使用javadoc工具生成信息，并输出到HTML文件中。

数据的输入和输出

  • 键盘输入
    • java.util.Scanner类创建对象，利用对象的方法：
    • nextInt()、nextDouble()、nextFloat()————输入某数据类型
    • [next()]、[nextLine()]————输入字符串
    • java.io.BufferedReader类(到输入输出时在接触)
      [next()]:结束符：空格、回车、TAB键
      [nextLine()]:结束符：回车
  • 控制台输出

    • System.out.print("Hello World");————输出字符串

    • System.out.println("Hello World");————输出字符串后换行

    • System.out.println(100);————输出整数后换行

    • 支持转义符

      • 换行符“\n”

      System.out.print("Hello kids!\n");

      System.out.print("You're welcome!");

      • 制表位“\t”

      System.out.print("Hello kids!\n");

      System.out.print("\tYou're welcome!");

关于基本数据类型的说明

  • Java基本数据类型的变量所占存储空间大小
    • 固定的，不随操作系统的变化而变化
    • 实现平台无关性、可移植性
  • 数值类型（6种）为有符号数。
  • 精度从低到高顺序

byte——short(char)——int——long——float——double

1——2——4——8——4——8

  • 布尔型占的存储空间大小没有明确指定。
    • 取值：true或false
    • 《Java虚拟机规范》规定：
    • Java语言表达式所操作的boolean值，在编译之后都使用Java虚拟机中的int数据类型来代替；
    • boolean数组将会编码成Java虚拟机的byte数组，每个boolean元素占8位。
    • 因此，一个boolean值占用内存四个字节的空间；但一个boolean数组中的每一个值占用内存1个字节的空间。
  • Java语法要求严格，每一种类型只能存储统一类型的数据，不允许乱放。
  • Java提供了方便之门，可以在不同的数据类型之间进行转换以满足需要。
  • 数据类型转换方式有哪些？
    • 自动类型转换
      • 默默进行的转换
        • int a=1;
        • long b=a;
      • 自动类型转换的条件
        • 两种类型相互兼容，如均为数值型（整型和浮点型）
      • 自动类型转换规则
        • 由低精度的数据类型向高精度的类型进行转换
          • byte——short(char)——int——long——float——double
          • 1——2——4——8——4——8
        • 在一个表达式中，如果某一个操作数为double类型，则整个表达式提升为double类型。
    • 强制类型转换
      • 强制类型转换可能会丢失精度，使用时需注意。

基本类型进阶和基本类型封装类、自动装拆箱

  • 每一种基本类型都有一个封装类
    • int——Integer
    • byte——Byte
    • short——Short
    • long——Long
    • double——Double
    • float——Float
    • char——Character
    • boolean——Boolean
  • 封装类的好处
    • 比基本类型提供更多的信息和功能
    • 如：各种类型的数据的最大及最小值

示例：Integer

  • 自动装箱与自动拆箱
    • int i=10;
    • Integer j=i; //自动装箱 autoboxing
    • int k=j; //自动拆箱 unboxing
    • 从JDK5.0版本开始的

运算符和表达式-算术、关系、逻辑

  • 算术运算符

    • + - * % ++ – -（单目减）

  • 关系运算符

    • < >= <= == != instanceeof

  • 逻辑运算符

    • && || ! ^ & |

  • 位运算

    • ~ & | >>（右移） <<（左移） >>>（无符号右移）

  • 赋值运算符

    • = += -= *= /= %=

  • 条件运算符

    • int flag=a>b?1:2;

  • &&和||

    • 采用短路规则
    • 示例
      • (a>3)&(b<=2) //若a>3为false，b<=2不需计算
      • (a>3)||(b<=2) //若a>3为true，b<=2不需计算

  • &和|

    • 不采用短路规则
    • 两个操作数均会被运算
      • (a>3)&(b<=2) //a>3和b<=2都需要计算
      • (a>3)|(b<=2) //a>3和b<=2都需要计算

一维数组

  • 数组类型时引用型数据类型
  • 由相同类型的若干项数据组成
    +public static void main(String[] args),其中：args是String类型的数组
    • int[] a=new int[10];
  • 数组元素在内存中占用连续的地址
  • 数组空间的静态性
    • 创建的时候指定长度（长度的数值可以是变量）
    • 一旦创建就不能修改数组的长度

int[] c = new int[12];

  • c是数组名

  • 获得数组的长度：c.length

  • 第一个数组元素的索引：0

  • 数组元素的访问：数组名[索引值]

  • 每个数组元素相当于普通的变量

    • c[0]=3;
    • c[0]+=5;

  • 数组的索引

    • 也称为下标
    • 必须数整数或整数表达式，例如：c[10]，c[i-3]

  • 一维数组的声明

    • 声明格式有两种：
      • 数组元素的数据类型[] 变量名;
      • 数组元素的数据类型 变量名[];
    • 示例：

    int[] c;

    int c[];

    String[] names;

    String names[];

  • 一维数组对象的创建

    • Java数组是对象，通过new关键字来创建.
    • 示例：
      `

int[] c; //声明时不必指定数组的大小
c=new int[12]; //创建对象，分配内存，元素赋初值可以简化成一行语句
int[] c=new int[12];
